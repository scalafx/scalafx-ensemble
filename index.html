ScalaFX - Ensemble.

ScalaFX Ensemble is a project which is inspired from http://jfxtras.org/ which is developed in JavaFX. Instead this project is developed in ScalaFX. We know that it looks slightly different from the JavaFx Ensemble we are working towards it thus making it as better as possible !!!!

Update 2019-11-09

Native installers for Windows, Mac OS, and Unix/Linux are available in ScalaFX Ensemble v.1.14.0. Downloads are available on the Releases page.
